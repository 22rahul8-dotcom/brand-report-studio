Brand Discovery — Paste any company's homepage URL and hit Discover Brand. The tool will extract their color palette, fonts, logo, and brand personality. Use this to understand a brand before generating a report.
Report Generator — Enter a company URL and describe what the report should cover (e.g. "annual growth, key metrics, product highlights"). The tool scrapes the site, applies the brand's own colors & fonts, and generates a downloadable HTML report. Add a GROQ_API_KEY env var to get AI-structured sections; otherwise it auto-structures the content.

Tools — Three raw scraping utilities. Scrape fetches any URL as markdown, HTML, links, or a screenshot. Extract JSON pulls structured data using a plain-English prompt or JSON schema. Batch Scrape processes multiple URLs at once — paste one URL per line.
